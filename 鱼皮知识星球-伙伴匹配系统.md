## 技术选型

**前端**

- Vue3
- Vant UI组件库
- Vite脚手架
- Axios请求库

**后端**

- Java SpringBoot
- MySQL
- MyBatis-Plus
- MyBatisX自动生成代码
- Redis缓存
- Redisson分布式锁
- EasyExcel数据导入
- SpringScheduler定时任务
- Gson
- 相似度匹配算法

**部署**

- serverless服务
- 云原生容器平台
